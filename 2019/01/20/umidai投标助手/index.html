umidai投标助手

Posted on 2019-01-20 | Edited on 2019-01-21 | In Tampermonkey

概述

获取页面投标倒计时, 自动投标.

安装

浏览器的选择

Chrome 或 Firefox

用户脚本管理器 ( Tampermonkey ) 安装

  1. Chrome 安装方式一:

    已爬墙的可在 google 应用商店安装: Tampermonkey .

  2. Chrome 安装方式二:

    1). 下载 Tampermonkey 4.7.63 CRX for Chrome or Chromium 至本地

    2). 打开浏览器扩展程序管理界面

    3). 将刚刚下载的 CRX 文件拖到浏览器内

  3. Firefox 安装方式:

    点击 进入 Firefox 扩展页面, 点击页面中的安装即可.

安装用户脚本

使用

打开有标的页面, 脚本自动运行.

# umidai
